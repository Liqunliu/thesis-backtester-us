## 烟蒂股深度价值分析框架

以资产清算价值为锚，通过T级资产分层评估静态价值，识别催化剂与价值陷阱，适用于PB极低但市场忽视的标的。

### 前置排除门控（必须首先执行）

烟蒂股的核心前提是"被市场抛弃的、经营困难的、但资产仍有清算价值的企业"。以下类型**不属于烟蒂股**，即使 PB 低也必须排除：

```
排除检查（任一命中 → is_cigar_butt = false，跳过后续全部分析）：

1. 持续盈利且增长的企业
   ├── 连续 3 年以上净利润为正且无明显下滑趋势
   └── 收入近 3 年无连续萎缩
   → 这是"低估值蓝筹"或"价值发现"，不是烟蒂股

2. 金融行业（银行/保险/证券）
   ├── 银行 PB < 1 是行业常态，不代表资产被低估
   ├── 金融企业资产负债表不适用 T 级清算分层
   └── 存款/保费负债不可简单按 100% 偿还计算清算价值
   → 应使用 PB-ROE 估值框架

3. 大市值央国企（市值 > 500 亿且控股股东为国资）
   ├── 国资背景企业不存在清算可能性
   └── 低 PB 更多反映市场风格而非资产低估
   → 应归入"纯硬收息"或"价值发现"流派
```

**输出要求**：若触发排除门控，`is_cigar_butt` 填 `false`，`liquidation_value` 填 0，并在分析中简要说明排除原因。

### 一、T级资产分层体系

将资产负债表中的资产按清算变现能力分为五层，逐层折价：

| 层级 | 资产类型 | 变现周期 | 折价率 | 代表科目 |
|------|----------|----------|--------|----------|
| **T0** | 现金及等价物 | 即时 | 0% | 银行存款、货币基金、交易性金融资产（货基类） |
| **T1** | 高流动性金融资产 | 1-30天 | 0-10% | 上市股票、债券、理财产品 |
| **T2** | 经营性流动资产 | 1-12月 | 10-30% | 应收账款（按账龄折价）、存货（按周转率折价） |
| **T3** | 固定资产 | >12月 | 30-50% | 厂房、设备、在建工程 |
| **T4** | 无形及难变现资产 | 极难变现 | 50-90% | 商誉、专利、长期待摊费用、递延税项 |

**T2层折价细则**：

| 科目 | 折价依据 | 估算方法 |
|------|---------|---------|
| 应收账款 | 账龄结构 | 1年内按90%计，1-2年按70%计，2年以上按30%计 |
| 存货 | 行业特性 | 快消品按80%计，制造业按60%计，地产按40-70%计 |
| 预付款项 | 可回收性 | 关联方按50%计，第三方按80%计 |

**清算价值计算公式**：

```
清算价值 = T0资产 x 100%
          + T1资产 x (90-100%)
          + T2资产 x (70-90%)
          + T3资产 x (50-70%)
          + T4资产 x (10-50%)
          - 全部负债（不折价，100%偿还）

安全边际 = 清算价值 / 市值
```

### 二、静态价值型三要求

烟蒂股投资必须同时满足以下三个条件：

**要求一：清算价值显著高于市值**

```
基本门槛：PB < 1（净资产 > 市值）
有意义门槛：PB < 0.5（净资产 > 2倍市值）
极端低估：PB < 0.3（净资产 > 3倍市值）

更精确的判定：
T级加权清算价值 > 市值 x 1.5（至少有50%清算安全边际）
```

**要求二：不持续烧钱（正经营现金流）**

```
必须满足：
├── 经营现金流为正（近3年中至少2年）
├── 不存在持续亏损烧光资产的趋势
└── 现有现金储备能支撑至少3年运营（假设零收入）

警示信号：
├── 经营现金流连续2年为负 → 高度警惕
├── 现金储备 < 1年固定开支 → 可能破产
└── 有息负债持续增加 → 资产在被稀释
```

**要求三：存在催化剂**

无催化剂的低估值股票可能永远低估（价值陷阱）。至少识别一种以下催化剂：

| 催化剂类型 | 说明 | 可验证性 |
|-----------|------|---------|
| **资产注入/重组** | 大股东注入优质资产，改善盈利能力 | 查公告、股东会议案 |
| **管理层变更** | 新管理层推动改革，释放资产价值 | 查人事变动公告 |
| **行业反转** | 行业周期触底，需求/价格开始回升 | 查行业数据、产能利用率 |
| **股东回购/增持** | 大股东或公司自身回购股份 | 查回购公告、持股变动 |
| **政策利好** | 行业政策转向支持 | 查政策文件 |
| **分红提升** | 提高派息比例，释放账上现金 | 查派息历史和政策 |

### 三、价值陷阱识别清单

以下信号出现时，需高度警惕"看似便宜但持续恶化"的价值陷阱：

```
财务恶化信号：
├── 收入连续3年下降 > 10%/年
├── 经营现金流转负且无改善迹象
├── 有息负债持续增加，杠杆率上升
├── 应收账款增速 > 收入增速（回款恶化）
└── 管理层频繁做大额资产减值

治理风险信号：
├── 大股东持续减持（套现离场）
├── 关联交易金额占比过高（>30%）
├── 频繁更换审计师
├── 管理层薪酬与业绩脱钩
└── 少数股东权益占比异常高

行业衰退信号：
├── 行业技术路线被替代（如胶卷被数码替代）
├── 政策永久性限制（非周期性调控）
├── 行业需求不可逆萎缩
└── 竞争格局恶化至全行业亏损
```

**价值陷阱排除决策树**：

```
发现PB极低标的
  ├── 有无正经营现金流？
  │   └── 否 → 高度警惕，需确认现金储备够撑3年
  ├── 收入是否持续萎缩？
  │   └── 是且>3年 → 确认行业是否不可逆衰退
  ├── 大股东是否在减持？
  │   └── 是 → 大概率价值陷阱，回避
  ├── 有无明确催化剂？
  │   └── 否 → 不投，等催化剂出现再考虑
  └── 以上均通过 → 可作为烟蒂股候选
```

### 四、REIT类期限结构估值

对于有明确到期日的资产（如信托、租赁权、特许经营权），采用期限结构估值：

```
估值公式：
标的价值 = 年金现值 + 到期残值现值

年金现值 = 年FCF x [(1-(1+r)^-n) / r]
到期残值现值 = 残值 x (1+r)^-n

其中：
  年FCF = 年度自由现金流
  r = 折现率（通常取8-10%）
  n = 剩余期限（年）
  残值 = 到期时可回收的资产净值
```

**债务偿还路径测算**：

```
评估框架：
├── 当前有息负债：[X]亿
├── 年FCF：[Y]亿
├── 年利息支出：[Z]亿
├── 可用于还债金额：Y - Z = [W]亿/年
├── 还清负债时间：X / W = [N]年
└── 判断：若N < 剩余期限的50%，则债务可控

分段收益评估：
├── 前N年：FCF主要用于还债，股东回报有限
├── 后(期限-N)年：债务还清，FCF全部归股东
└── 到期时：回收残值（需评估资产状况）
```

### 五、烟蒂股投资纪律

**买入纪律**：

| 条件 | 阈值 | 必要性 |
|------|------|--------|
| PB | < 0.5 | 必须 |
| T级清算价值/市值 | > 1.5 | 必须 |
| 经营现金流 | 正值（近3年2年以上） | 必须 |
| 催化剂 | 至少识别1种 | 必须 |
| 价值陷阱排除 | 通过排除检查 | 必须 |

**卖出纪律**：

| 流派 | 买入标准 | 修复目标 | 卖出触发 |
|------|---------|---------|---------|
| 烟蒂股 | PB < 0.3 | PB → 0.6 | PB >= 0.6 减仓50% |
| 烟蒂股 | PB < 0.5 | PB → 0.8 | PB >= 0.8 减仓50% |

**仓位限制**：

```
单只烟蒂股仓位上限：5%（风险较高，低于正常标的的8-15%）
烟蒂股组合占比上限：20%（组合层面控制风险）
持有期限预期：1-3年（等待催化剂兑现）
```
