## 轻资产商业模式分析框架

识别和评估具有"一次性投入、边际成本趋零、垄断定价"特征的轻资产模式，重点关注FCF质量、营运资本效率和护城河持续性。

### 前置排除门控（必须首先执行）

以下行业的商业模式本质上依赖资产负债表规模（高杠杆经营），**不属于轻资产模式**：

```
排除检查（任一命中 → is_light_asset = false，跳过后续全部分析）：

1. 银行/保险/证券/多元金融
   ├── 银行靠存贷利差（资产负债表驱动），资产负债率 80-95% 是行业常态
   ├── 保险靠浮存金投资（保费=负债），杠杆经营本质
   └── 证券靠自营+两融（资本金驱动）
   → 属于"重资产负债表"模式，与轻资产完全相反

2. 房地产/建筑施工
   ├── 高土地储备+高预收+高负债的资本密集型模式
   └── CAPEX/收入 远超 10%

3. 资产负债率 > 70% 且非平台型企业
   └── 高杠杆本身说明企业依赖外部资金而非自身造血
```

**输出要求**：若触发排除门控，`is_light_asset` 填 `false`，`recurring_revenue_pct` 正常计算填写，并简要说明排除原因。

### 一、轻资产模式特征识别

符合以下3条中至少2条，可判定为轻资产模式：

| 特征 | 判定标准 | 说明 |
|------|---------|------|
| **边际成本趋零** | 新增客户/交易的增量成本<收入的10% | IT系统、平台、特许经营权等 |
| **高利润率** | 毛利率>60% 或 净利率>25% | 无需大量原材料或人工 |
| **低资本支出** | CAPEX/收入<10%（维护期） | 建设期可能较高，但运营期很低 |

**三类典型轻资产模式**：

```
类型一：垄断IT/数据平台
├── 特征：数据中心一次性CAPEX，后续维护成本极低
├── 护城河：数据壁垒 + 切换成本 + 政策准入
├── 典型行业：航空订座系统(GDS)、金融交易所、征信系统
└── 关键指标：FCF约等于净利润（折旧后几乎无额外支出）

类型二：网络效应平台
├── 特征：用户越多价值越大，赢家通吃
├── 护城河：网络效应 + 用户习惯 + 数据飞轮
├── 典型行业：社交平台、支付网络、电商市场
└── 关键指标：用户增长率、ARPU、获客成本趋势

类型三：品牌/特许经营
├── 特征：品牌溢价带来定价权，轻资产运营
├── 护城河：品牌心智 + 渠道网络 + 配方/工艺独占
├── 典型行业：奢侈品、烈酒、特许经营餐饮
└── 关键指标：品牌溢价率、特许权使用费收入占比
```

### 二、估值方法（FCF/利润率为核心）

轻资产模式的估值以FCF为核心，PE可作为辅助参考（因利润质量通常较高）。

#### 2.1 FCF质量验证

```
轻资产模式的FCF质量检验：

经营现金流 / 净利润 > 100%（应大于1，因折旧低）
├── 若 > 120% → 现金流质量极佳，利润全为真金白银
├── 若 80-100% → 正常范围
└── 若 < 80% → 异常，需查是否有隐藏的资本支出或应收恶化

FCF / 净利润 > 80%（维持性CAPEX应很低）
├── 若 > 90% → 典型轻资产特征
├── 若 70-90% → 可能处于升级/扩张期
└── 若 < 70% → 可能并非真正的轻资产模式

维持性CAPEX / 折旧 < 80%（折旧已充分覆盖资本支出）
```

#### 2.2 核心估值指标

| 指标 | 计算方法 | 轻资产合理区间 | 说明 |
|------|---------|--------------|------|
| **FCF/市值** | FCF / 总市值 | >8%为有吸引力 | 核心指标 |
| **剔除净现金FCF倍数** | (市值-净现金) / FCF | <8倍为低估 | 轻资产常有大量现金 |
| **净现金/市值** | 净现金 / 总市值 | >50%为极佳 | 轻资产公司现金积累快 |
| **ROIC** | 税后经营利润 / 投入资本 | >20%为优秀 | 衡量资本效率 |
| **净利率** | 净利润 / 收入 | >25%为轻资产特征 | 利润率稳定性是关键 |

#### 2.3 估值决策树

```
轻资产标的估值流程：

步骤1：确认轻资产特征
  ├── 毛利率>60% 或 净利率>25%？
  ├── CAPEX/收入<10%（运营期）？
  ├── FCF/净利润>80%？
  └── 满足2/3 → 确认为轻资产模式

步骤2：计算核心指标
  ├── FCF/市值 = [X]%
  ├── 剔除净现金FCF倍数 = [Y]倍
  ├── 净现金/市值 = [Z]%
  └── ROIC = [W]%

步骤3：估值判定
  ├── FCF/市值>8% 且 FCF倍数<8倍 → 低估，可建仓
  ├── FCF/市值>15% 且 净现金/市值>50% → 显著低估
  └── FCF/市值<5% → 高估或增长预期过高，谨慎
```

### 三、营运资本效率分析

轻资产模式的营运资本效率是竞争优势的体现：

| 指标 | 计算方法 | 优秀标准 | 意义 |
|------|---------|---------|------|
| **应收周转天数** | 应收账款 / (收入/365) | <60天 | 定价权和议价能力 |
| **存货周转天数** | 存货 / (成本/365) | <30天或无存货 | 轻资产通常低存货 |
| **应付周转天数** | 应付账款 / (成本/365) | >90天 | 占用上游资金的能力 |
| **现金转换周期** | 应收+存货-应付（天数） | <0天（负值最佳） | 负值意味着占用他人资金经营 |

```
营运资本效率判定：

现金转换周期 < 0天
├── 说明：公司用上下游的钱做生意，自身资金占用极低
├── 典型：预收模式（先收钱后服务）、账期优势（慢付款快收款）
└── 投资意义：经营规模扩大不需要额外营运资本投入

现金转换周期 > 60天
├── 说明：营运资本占用较高，可能不是典型轻资产
├── 需要检查：是否有大额应收、存货积压
└── 投资意义：增长需要持续投入营运资本，FCF可能低于利润
```

### 四、与重资产模式的对比

| 维度 | 轻资产模式 | 重资产模式 |
|------|-----------|-----------|
| **CAPEX/收入** | <10%（运营期） | >20% |
| **折旧/净利润** | <30% | >50%（常扭曲利润） |
| **FCF/净利润** | >80% | 可能<50% |
| **ROE驱动** | 高利润率驱动 | 高杠杆驱动 |
| **增长方式** | 用户/流量增长，边际成本低 | 产能扩建，资本密集 |
| **经济周期敏感度** | 较低（刚性需求/垄断） | 较高（产能过剩风险） |
| **估值偏好** | PE/FCF均可（利润质量高） | 强制FCF（PE被折旧扭曲） |
| **现金积累** | 快速积累，常有大量净现金 | 持续回投CAPEX，现金有限 |

### 五、轻资产模式风险清单

```
风险一：护城河消失
├── 技术迭代：核心系统被新技术替代（如云计算替代本地部署）
├── 政策变化：垄断地位被监管打破（如反垄断、开放准入）
├── 竞争进入：新玩家绕过壁垒（如开源替代商业软件）
└── 监控指标：市场份额变化、新进入者数量、政策动向

风险二：增长天花板
├── 用户渗透率接近饱和
├── ARPU增长停滞
├── 需要进入新市场/业务线（可能不擅长）
└── 监控指标：收入增速趋势、同店/同用户增长

风险三：利润率下降
├── 竞争加剧导致定价权丧失
├── 成本端上升（如人工成本、合规成本）
├── 客户集中度过高（大客户议价）
└── 监控指标：毛利率/净利率趋势、前5大客户占比

风险四：现金分配不当
├── 管理层囤积现金不回馈股东
├── 盲目多元化（轻资产公司收购重资产业务）
├── 高溢价并购导致商誉风险
└── 监控指标：派息率、回购金额、并购历史
```
