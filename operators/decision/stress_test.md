## 极端情景压力测试框架

设定"连续三年每年衰退10%"的极端悲观情景，通过三项安全测试验证标的在最恶劣情况下的生存能力和投资安全性。

### 一、极端悲观情景设定

统一采用"连续三年衰退10%"作为压力测试基准：

```
情景名称："连续三年衰退10%"

假设条件：
├── Year 0（当前）：净利润 = P 亿元
├── Year 1：净利润 = P x 0.90
├── Year 2：净利润 = P x 0.81
├── Year 3：净利润 = P x 0.729
└── 三年累计利润 = P x (0.90 + 0.81 + 0.729) = P x 2.439

同步假设：
├── 经营现金流同步下降10%/年
├── 资本支出维持不变（保守假设）
├── 股息政策不变（测试可持续性）
└── 不考虑任何一次性收益
```

### 二、三项安全测试

#### 测试1：利润覆盖测试

**核心问题**：即使连续三年衰退，累计利润能否覆盖扣除现金后的市值？

```
计算公式：
三年累计利润 = 当前净利润 x 2.439

判定标准：
三年累计利润 > 剔除净现金后市值（EV）

其中：
EV = 市值 - 净现金
净现金 = 现金及等价物 - 有息负债
```

| 结果 | 判定 | 含义 |
|------|------|------|
| 三年累计利润 > EV | 通过 | 即使连续衰退，3年利润即可收回投资 |
| 三年累计利润 < EV 但 > EV x 0.7 | 边缘 | 安全边际不足，需谨慎 |
| 三年累计利润 < EV x 0.7 | 不通过 | 极端情景下投资回收期过长 |

#### 测试2：现金覆盖测试

**核心问题**：账上现金加三年经营现金流，能否覆盖市值？

```
计算公式：
当前现金存量 = 现金及等价物（T0资产）
三年累计经营现金流 = 当前经营现金流 x 2.439（同比例衰退）
三年累计FCF = 三年累计经营现金流 - 3年维持性CAPEX

判定标准：
当前现金存量 + 三年累计FCF > 市值
```

| 结果 | 判定 | 含义 |
|------|------|------|
| 现金 + 三年FCF > 市值 | 通过 | 现金安全垫极厚 |
| 现金 + 三年FCF > 市值 x 0.8 | 边缘 | 基本安全，需监控 |
| 现金 + 三年FCF < 市值 x 0.8 | 不通过 | 现金保护不足 |

#### 测试3：分红可持续性测试

**核心问题**：即使利润连续衰退三年，当前分红水平能否维持？

```
计算公式：
Year 3 净利润 = 当前净利润 x 0.729
Year 3 派息支出 = 当前年度股息总额（假设不变）
Year 3 分红支付率 = Year 3 派息支出 / Year 3 净利润
Year 3 FCF覆盖率 = Year 3 FCF / Year 3 派息支出
```

| 指标 | 安全阈值 | 危险阈值 |
|------|---------|---------|
| Year 3 分红支付率（股息/净利润） | < 70% | > 100% |
| Year 3 FCF覆盖率（FCF/股息） | > 1.5倍 | < 1.0倍 |
| Year 3 股息率（基于当前股价） | > 3% | < 2% |

### 三、综合判定

| 通过测试数 | 综合评级 | 操作建议 |
|-----------|---------|---------|
| 3项全部通过 | 极度安全 | 可重仓（核心标的15%） |
| 2项通过 | 基本安全 | 可正常建仓（卫星标的8%） |
| 1项通过 | 安全边际不足 | 仅限观察仓位（2%以下） |
| 0项通过 | 不安全 | 不建仓，排除 |

### 五、补充情景（可选）

当标的处于周期行业时，可追加以下补充情景测试：

| 补充情景 | 假设 | 核心验证 |
|---------|------|---------|
| 行业产能过剩 | 产品价格降至历史10%分位 | FCF是否仍为正 |
| 需求断崖 | 收入下降30%（一次性冲击） | 现金储备能否撑过低谷 |
| 汇率冲击 | 本币贬值20% | 外币负债是否导致资不抵债 |
| 政策冲击 | 核心业务受限（税率/牌照） | 替代收入能否覆盖运营成本 |

对于周期股，需同时通过基准情景（三年-10%）和至少一项补充情景，方可视为安全。
