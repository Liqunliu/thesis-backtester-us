## 分析目标

评估公司在行业中的竞争地位、护城河强度与持久性、市场份额变化趋势、定价能力，以及行业竞争格局的演变方向。本算子侧重前瞻性判断——不仅看"现在强不强"，更要看"未来能否持续强"。

## 分析框架

### 一、行业定位

基于公司收入规模、市值及行业竞争格局判断市场地位：

| 定位 | 判定标准 | 典型特征 |
|------|---------|---------|
| 龙头 | 行业收入/利润排名前 3 | 具有定价权，品牌辨识度高，能引领行业标准 |
| 第二梯队 | 行业排名 4-10 | 有一定竞争力，在细分领域可能领先 |
| 一般 | 中等规模，排名 10-20 | 无明显差异化优势，跟随行业走势 |
| 边缘化 | 小市值、份额持续下降 | 有被整合或淘汰的风险 |

**操作步骤：**
1. 调用 `query_market_context(info_type="industry_overview")` 获取行业概况
2. 比较公司营收与行业内主要竞争者的营收排名
3. 结合市值排名交叉验证（有时收入高但市值低，说明市场对其持续性存疑）

### 二、波特五力分析

对以下五个维度逐一评估，每项给出 高/中/低 评级并附一句话证据：

| 竞争力量 | 评级 | 关键判断标准 |
|---------|------|------------|
| **现有竞争者的对抗强度** | 高/中/低 | CR3 < 30% → 高（分散竞争）；CR3 > 60% → 低（寡头格局稳定）；同行业价格战频率 |
| **新进入者威胁** | 高/中/低 | 资本开支门槛 < 10亿 → 高；需牌照/审批 → 低；技术壁垒判断 |
| **替代品威胁** | 高/中/低 | 是否存在功能替代的新技术/新模式？替代品价格是否更优？渗透速度如何？ |
| **买方议价能力** | 高/中/低 | 前5大客户收入占比 > 50% → 高；产品差异化程度低 → 高；to-C且品牌强 → 低 |
| **供应商议价能力** | 高/中/低 | 关键原材料是否被少数供应商垄断？是否有长协锁价？可替代供应商数量 |

**五力综合评判：**
- 5项中 ≥3项 为"低" → 行业竞争环境友好，利于龙头持续盈利
- 5项中 ≥3项 为"高" → 行业竞争激烈，即使龙头也难以持续超额利润
- 重点关注变化方向：任何一项从"低"向"高"转变，都是护城河侵蚀的早期信号

### 三、定价能力测试

定价能力是护城河的终极验证——有护城河的公司应当能在提价后维持销量。

**测试方法：**

1. **历史提价记录**：过去 3 年内公司是否有主动提价行为？
   - 调用 `query_market_context(info_type="news")` 搜索提价相关信息
   - 查看毛利率变化趋势：毛利率逐年提升可能反映隐性提价

2. **量价关系验证**：
   - 提价后销量是否保持稳定或仅小幅下降（<5%）→ 强定价权
   - 提价后销量明显下降（>10%）→ 弱定价权，产品可替代性强
   - 从未提价或持续降价保份额 → 无定价权

3. **定价能力评级标准**：

| 评级 | 判定条件 |
|------|---------|
| 强 | 过去 3 年有成功提价记录，且销量未受明显影响；或毛利率持续提升 >2pp |
| 中 | 能随成本上涨传导提价，但无法超越成本涨幅；毛利率基本稳定（±1pp） |
| 弱 | 无法转嫁成本上涨，或提价导致份额流失；毛利率持续下降 |

### 四、市场份额趋势判断

**核心方法：公司收入增速 vs 行业收入增速的对比**

计算近 3 年（推荐使用 3 年 CAGR，避免单年波动干扰）：
- 公司营收 CAGR
- 行业整体营收 CAGR（通过行业概况数据或可比公司估算）

| 对比结果 | 份额趋势 |
|---------|---------|
| 公司 CAGR > 行业 CAGR 超过 3个百分点 | 抢占份额 |
| 差值在 ±3个百分点以内 | 稳定 |
| 公司 CAGR < 行业 CAGR 超过 3个百分点 | 流失份额 |

**辅助验证信号：**
- 抢占份额的佐证：主要竞争者份额下降、公司产能利用率上升、新客户/新区域拓展
- 流失份额的佐证：竞争者增速更快、客户集中度提高（被迫依赖大客户）、产能利用率下降

### 五、护城河类型与持久性评估

不仅要识别护城河类型，更要评估其持久性和变化方向。对每种护城河类型，需要给出具体的量化证据：

#### 5.1 规模护城河

| 评估维度 | 量化标准 | 数据来源 |
|---------|---------|---------|
| 营收规模差距 | 公司营收 / 第二名营收 > 1.5x → 规模优势明显 | income |
| 单位成本优势 | 公司毛利率 - 行业中位数 > 5pp，且营收规模领先 → 规模效应转化为成本优势 | fina_indicator |
| 产能利用率 | 产能利用率 > 80% 且高于同行 → 规模效率高 | 行业数据 |
| 持久性判断 | 行业是否仍在扩产？若竞争者大规模扩产（行业产能增速 > 需求增速），规模优势可能被稀释 | — |

#### 5.2 品牌护城河

| 评估维度 | 量化标准 |
|---------|---------|
| 定价溢价 | 同品类产品价格 > 竞品 15% 以上，且销量未受影响 |
| 毛利率优势 | 毛利率持续 > 行业平均 10pp 以上 |
| 品牌稳定性 | 毛利率近 5 年波动 < 3pp → 品牌认知稳定；波动 > 5pp → 品牌溢价不稳定 |
| 侵蚀信号 | 营销费用率持续上升但收入增速放缓 → 品牌需要更多投入维持，可能在侵蚀中 |

**注意：** 仅凭毛利率高不能断定有品牌护城河。需同时满足"高毛利率 + 定价溢价 + 品牌认知稳定"三个条件。

#### 5.3 网络效应护城河

| 评估维度 | 量化标准 |
|---------|---------|
| 用户增长趋势 | 活跃用户/交易量近 3 年 CAGR > 15% → 网络仍在扩张 |
| 网络密度 | 双边平台：买卖双方比例是否健康？单边过剩 → 网络效应减弱 |
| 单用户价值 | ARPU（每用户收入）是否随用户规模增长而提升？是 → 正向网络效应 |
| 侵蚀信号 | 用户增长停滞但获客成本持续上升 → 网络效应见顶 |

#### 5.4 转换成本护城河

| 评估维度 | 量化标准 |
|---------|---------|
| 客户留存率 | 留存率 > 90% → 转换成本高；留存率 < 80% → 转换成本低 |
| 合同锁定 | 平均合同期限 > 3 年 → 强锁定；年度续约制 → 弱锁定 |
| 系统嵌入深度 | 产品是否嵌入客户核心业务流程？替换需要多长时间和多大成本？ |
| 侵蚀信号 | 合同续约率连续 2 年下降、竞品推出低成本迁移方案 |

#### 5.5 牌照/特许经营护城河

| 评估维度 | 量化标准 |
|---------|---------|
| 牌照稀缺性 | 全行业持牌机构数量 < 20 → 高稀缺；> 100 → 低稀缺 |
| 续期风险 | 牌照是否有到期日？续期是否需要满足额外条件？ |
| 政策依赖度 | 收入中多大比例依赖于牌照赋予的特许权？与 policy_risk 算子交叉验证 |
| 侵蚀信号 | 监管放松准入（如发放更多牌照）、行业改革试点（如电力市场化） |

#### 5.6 资源垄断护城河

| 评估维度 | 量化标准 |
|---------|---------|
| 资源储量 | 已探明储量可供开采年限 > 15 年 → 长期优势 |
| 资源品位 | 矿石品位/资源质量是否显著优于行业平均？ |
| 替代风险 | 是否存在替代资源或回收技术？替代品成本差距多大？ |

#### 护城河持久性综合判断

```
加固中：护城河指标近 3 年持续改善（如份额上升 + 毛利率提升 + 定价权增强）
稳定  ：核心指标无明显变化，竞争格局未出现新变量
侵蚀中：出现 2 项以上侵蚀信号（如新进入者抢份额、定价权减弱、客户留存下降）
不适用：公司无明显护城河
```

### 六、行业生命周期判定

**基础判定标准：**

| 阶段 | 收入增速 | 渗透率 | 竞争格局 |
|------|---------|--------|---------|
| 增长期 | > 15% | < 50% | 参与者增多，格局未定 |
| 成熟期 | 5%-15% | > 50% | CR3 稳定，进入壁垒高 |
| 衰退期 | < 5% 或负增长 | — | 被新技术/模式替代 |
| 周期性 | 随经济周期大幅波动（标准差 > 10pp） | — | 与宏观/商品价格强相关 |

**周期性行业特别规则：**
- 对于周期性行业（化工、钢铁、航运、养殖等），**必须使用 5 年平均收入增速**判定生命周期阶段，不可使用单年增速
- 原因：周期底部单年增速可能为负增长，被误判为"衰退"；周期顶部单年增速可能 >30%，被误判为"增长期"
- 计算方法：取最近 5 个完整年度的收入，计算 5 年 CAGR 作为判定依据

### 七、竞争响应分析

评估公司的竞争优势被追赶的风险：

**成本优势的可持续性：**
- 成本优势来源是规模（可追赶）还是资源禀赋（难追赶）？
- 竞争者是否在扩产？新产能投产后的预期成本差距是否缩小？
- 技术进步（如自动化、新工艺）是否可能让小规模竞争者也达到类似成本？
- **量化参考**：若竞争者 3 年内新增产能 > 公司现有产能的 50%，成本优势被侵蚀的概率较高

**品牌优势的可持续性：**
- 品牌认知是否稳定？消费者调研/搜索指数趋势如何？
- 是否出现新锐品牌快速崛起（尤其在线上渠道）？
- 品牌老化信号：核心消费群体年龄上移、年轻用户渗透率下降
- **量化参考**：广告费用率持续上升 >2pp 但市占率未提升 → 品牌竞争力下降

**技术优势的可持续性：**
- 技术迭代周期多长？专利保护期还剩多久？
- 竞争者研发投入趋势：若竞争者研发费用率 > 公司 + 2pp，可能在追赶
- 是否有颠覆性技术路线出现（如锂电 vs 钠电、燃油 vs 电动）？

### 八、分析输出模板

完成以上分析后，按以下格式输出结论：

1. **行业定位**：[龙头/第二梯队/一般/边缘化]，排名依据：[收入/市值/细分市场]
2. **五力分析摘要**：[逐项评级]，综合判断竞争环境是否友好
3. **定价能力**：[强/中/弱]，证据：[提价记录/毛利率趋势]
4. **市场份额趋势**：[抢占/稳定/流失]，公司 3 年 CAGR [X%] vs 行业 [Y%]
5. **护城河类型**：[类型]，持久性：[加固中/稳定/侵蚀中]
6. **行业生命周期**：[增长/成熟/衰退/周期性]（周期性行业注明 5 年平均增速）
7. **竞争响应风险**：[竞争者追赶的最大威胁是什么？预计多久可能追平？]

## 注意

- 银行、保险等牌照型企业天然有强护城河，但需注意监管放松准入的风险
- 周期性行业的龙头在周期底部可能被误判为"衰退"——务必用 5 年平均增速判断
- 护城河强度应看 3-5 年视角，不是当期财报能完全反映的
- 与 `policy_risk` 算子交叉参考：牌照/特许经营类护城河的持久性高度依赖政策稳定性
- 消费品行业的品牌护城河需要区分"真品牌"（消费者主动选择）和"渠道依赖"（靠铺货量取胜）
