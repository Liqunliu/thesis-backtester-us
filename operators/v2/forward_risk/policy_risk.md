## 政策转向风险评估

A股投资的核心风险维度。中国市场政策驱动特征极强，单一政策可在短期内摧毁一个行业的盈利基础。本算子对政策风险进行系统性、多维度评估。

### 前置要求：必须调用工具获取最新政策信号

```
强制操作（分析开始前必须执行）：
├── 调用 query_market_context(info_type="news") 获取最新新闻
│   └── 从新闻中提取与该公司/行业相关的政策信号
├── 不得仅依赖背景知识判断政策方向
│   └── 政策变化速度极快，背景知识可能过时
└── 如工具返回无相关新闻，明确标注"未获取到近期政策信号，以下基于历史政策分析"
```

### 一、国家战略对齐度评估

评估公司业务与国家战略优先级的对齐程度。战略对齐度直接决定政策风险的方向性：对齐度高的公司享受政策红利，对齐度低或位于打击方向的公司面临政策逆风。

**当前国家战略优先级清单**：

| 战略方向 | 核心领域 | 政策含义 |
|---------|---------|---------|
| 新质生产力 | AI、量子计算、生物制造、商业航天 | 最高优先级扶持 |
| 中国制造2025 | 半导体、高端装备、新材料、新能源 | 产业政策+财政补贴+政府采购倾斜 |
| 双循环 | 内需消费升级、进口替代、供应链安全 | 内需型企业受益，外需依赖型承压 |
| 共同富裕 | 收入分配、反垄断、教育公平 | 高利润率平台/垄断企业面临再分配压力 |
| 碳达峰碳中和 | 新能源、储能、碳交易 | 传统高耗能行业产能受限 |
| 数据安全 | 数据治理、个人信息保护 | 数据密集型企业合规成本上升 |

**对齐度评级**：

```
判断流程：
├── 核心战略（最高保护）：公司主营业务直接服务于上述战略核心领域
│   └── 特征：获得专项基金、税收优惠、政府采购定向支持
├── 支撑配套（间接受益）：为战略产业提供配套服务/产品
│   └── 特征：下游客户集中在战略行业
├── 中性（不在政策视野内）：传统消费、一般制造等
│   └── 特征：既无明显扶持也无打压迹象
└── 潜在打击对象（高风险）：业务模式与当前政策导向冲突
    └── 特征：媒体批评增多、监管约谈、行业整顿信号
```

### 二、A股历史政策冲击案例库

**必须判断**：该公司所在行业是否在过去十年经历过重大政策冲击？如果是，描述冲击内容、持续时间、对行业盈利的量化影响。

| 行业 | 时间 | 政策事件 | 代表公司 | 冲击幅度 |
|------|------|---------|---------|---------|
| 教育培训 | 2021 | 双减政策：禁止K12学科类培训机构上市融资、营利 | 好未来、新东方 | 收入近乎归零，市值跌幅>90% |
| 互联网平台 | 2021-2022 | 反垄断：罚款+业务限制+数据安全审查 | 阿里巴巴、腾讯、滴滴 | 市值腰斩，业务模式被迫调整 |
| 房地产 | 2020-2023 | 三道红线+限购限贷+预售资金监管 | 恒大、碧桂园、融创 | 多家龙头暴雷，行业销售额腰斩 |
| 游戏 | 2021 | 版号冻结+未成年人防沉迷 | 腾讯、网易 | 新游戏收入断档，未成年人收入归零 |
| 医药 | 2018至今 | 集采(VBP)：药品/耗材大幅降价 | 仿制药企、医疗器械 | 中标品种降价60-90%，利润率压缩至个位数 |
| 新能源 | 2019 | 光伏补贴退坡 | 光伏组件企业 | 毛利率短期承压，行业洗牌 |
| 白酒 | 2012-2014 | 八项规定+限制三公消费 | 贵州茅台、五粮液 | 高端白酒量价齐跌，行业调整2-3年 |
| 影视传媒 | 2018 | 范冰冰事件后税务整顿+限薪令 | 华谊兄弟、唐德影视 | 行业税后利润大幅下降，项目开工锐减 |

**关键教训**：

- 政策冲击通常没有预警，公告即执行
- 冲击幅度远超市场预期（教培行业从千亿赛道到接近清零）
- 恢复期极长（地产至今未见底，教培永久性转型）
- 越是高利润、高社会关注度的行业，越容易成为政策目标

### 三、行业政策依赖度分级

| 依赖度 | 典型行业 | 政策传导机制 |
|--------|---------|-------------|
| 极高 | 地产、教培、医药集采、新能源补贴 | 政策直接决定行业存亡或定价权 |
| 高 | 银行（利率政策）、基建（财政政策）、军工、烟草 | 政策决定收入规模或利润率 |
| 中高 | 互联网平台、游戏、传媒 | 牌照/审批制度制约增长空间 |
| 中 | 消费（消费刺激）、出口（关税/汇率）、汽车 | 政策影响需求侧，但非决定性 |
| 低 | 日常消费品、公用事业、物业服务 | 刚需属性，政策影响有限 |

### 四、当前政策方向判断

基于工具返回的新闻信号和行业认知，判断当前政策环境：

```
政策方向判断矩阵：
├── 强力鼓励/支持：
│   ├── 信号：专项基金设立、税收减免、政府工作报告重点提及
│   ├── 典型：AI算力、半导体设备、新能源车
│   └── 风险：支持政策可能退坡，需评估补贴依赖度
├── 温和支持/中性：
│   ├── 信号：无明显限制，正常监管
│   └── 风险：可能因社会事件突然转向
├── 收紧/限制中：
│   ├── 信号：频繁约谈、出台限制性规定、媒体批评
│   ├── 典型：地产调控、互联网反垄断
│   └── 风险：已在冲击过程中，需评估是否见底
├── 方向不明/转向中：
│   ├── 信号：政策信号矛盾、不同部门口径不一
│   └── 风险：最高不确定性阶段
└── 从收紧转向宽松：
    ├── 信号：调控政策边际放松、救市措施出台
    ├── 典型：2022年底地产"三支箭"、2023年互联网平台"绿灯"
    └── 机会：政策底往往先于市场底
```

### 五、收入政策敏感度分析

利用分业务收入数据(fina_mainbz)，估算直接受政策影响的收入占比：

```
计算方法：
1. 列出公司各业务板块收入（来自主营业务构成数据）
2. 逐一判断每个板块的政策敏感度
3. 加权计算整体政策敏感收入占比

示例：
某医药公司：
├── 仿制药（集采覆盖品种）：收入40亿，占比60% → 政策高敏感
├── 创新药（未纳入集采）：收入15亿，占比22% → 政策中敏感
├── 原料药出口：收入12亿，占比18% → 政策低敏感（但受关税影响）
└── 政策敏感收入占比 = 60% × 1.0 + 22% × 0.5 + 18% × 0.3 = 76.4%

政策敏感度评级：
> 70%：极高敏感——政策变动直接威胁公司生存
50-70%：高敏感——需密切关注政策方向
30-50%：中等敏感——有缓冲但不可忽视
< 30%：低敏感——政策非主要风险因素
```

### 六、政府补助与补贴依赖度分析

```
数据来源：利润表"其他收益"或"营业外收入"中的政府补助项
计算：政府补助占净利润比例 = 政府补助 / 归母净利润 × 100%

依赖度评级：
├── > 50%：极度依赖——扣除补贴后可能亏损，风险极高
├── 20-50%：高度依赖——补贴退坡将严重影响盈利能力
├── 10-20%：中度依赖——需关注补贴可持续性
├── < 10%：低依赖——补贴是锦上添花
└── 0%或接近0%：无依赖

补贴可持续性判断：
├── 产业扶持类补贴（如芯片、新能源）：与政策周期绑定，通常3-5年
├── 税收返还类补贴（如西部开发、高新技术）：相对稳定但有到期风险
├── 项目类补贴（如科研项目）：一次性，不可持续
└── 出口退税类补贴：受贸易政策影响，可能调整
```

**重点警示**：政府补助占净利润 > 20% 的公司属于高政策依赖型。需额外检查：
- 补贴来源是否集中在单一政府项目/政策？
- 同行业公司是否也享受类似补贴（行业性 vs 个案性）？
- 补贴政策是否有明确到期日？

### 七、地方政府财政依赖度分析

```
评估维度：
├── 收入来源结构：
│   ├── 政府采购/工程合同收入占比？
│   ├── 应收账款中政府客户占比？
│   └── 前五大客户中是否有政府或政府平台公司？
├── 地方政府财政健康度：
│   ├── 所在地财政自给率（一般公共预算收入/支出）
│   ├── 地方政府显性+隐性债务率
│   ├── 土地财政依赖度（土地出让金占财政收入比）
│   └── 是否已出现公务员降薪、城投非标逾期等信号？
└── 回款风险评估：
    ├── 应收账款周转天数是否逐年拉长？
    ├── 政府类应收是否存在大额坏账计提？
    └── 现金流量表经营活动现金流是否持续低于净利润？
```

**核心逻辑**：2023年以来地方政府债务约束持续收紧，依赖政府支出的企业（环保、园林、智慧城市、地方基建）面临系统性的回款延迟和订单萎缩风险。即使利润表显示盈利，现金流可能已经恶化。

### 八、央国企 vs 民企政策韧性差异

```
政策冲击抵御能力对比：

央国企优势：
├── 政策制定参与者——通常提前知晓政策方向
├── 资源获取优先——在收紧周期更易获得信贷、牌照、项目
├── 隐性政府担保——极端情况下不会被允许破产
├── 政策执行成本由国家承担——如银行让利实体经济
└── 但：也承担政策性亏损义务（如保供稳价、援助任务）

民企劣势：
├── 政策被动接受者——反应时间短
├── 融资渠道收窄——收紧周期首先被抽贷
├── 无隐性担保——极端情况下可被清退出局
├── 合规成本相对更高——同等监管力度下负担更重
└── 但：决策灵活，转型速度快，部分民企在政策冲击后快速转型成功

韧性评级：
├── 强韧性：央国企 + 核心战略行业
├── 中韧性：央国企非核心行业 / 民企龙头（行业地位稳固）
├── 弱韧性：民企 + 高政策敏感行业 + 非龙头地位
└── 极弱韧性：民企 + 处于政策打击方向 + 高杠杆
```

### 九、政策冲击极端情景估算

**必须完成**：假设该公司所在行业遭遇类似历史案例的政策冲击，估算对盈利的影响。

```
估算框架：
1. 确定政策敏感业务线收入占比 → A%
2. 参考历史案例中该类政策对收入的冲击幅度 → B%
   ├── 全面禁止型（如教培双减）：收入影响 80-100%
   ├── 价格管制型（如医药集采）：收入影响 40-70%
   ├── 监管罚款型（如反垄断罚款）：一次性影响 + 业务调整成本
   ├── 准入限制型（如游戏版号）：增长停滞但存量业务保留
   └── 补贴退坡型（如新能源补贴）：毛利率下降 5-15pp
3. 极端情景盈利影响 = A% × B% × 当前净利润
4. 评估公司是否有足够现金/资产应对（查看资产负债表）

输出：
├── 极端情景下盈利影响占比：[ ]%
├── 公司现金储备能否覆盖 2 年冲击期：是/否
├── 是否存在业务转型空间：是（说明方向）/ 否
└── 综合生存概率判断：高/中/低
```

### 注意事项

- **必须调用 query_market_context(info_type="news")**获取最新政策信号，不得省略
- 政策影响往往滞后 6-12 个月才反映到财报，但股价反应通常在政策公布当日
- 关注"政策底"与"市场底"的时滞——政策转向宽松后，行业基本面可能仍在恶化
- 同一行业内不同公司受政策冲击程度可能差异巨大（龙头 vs 中小企业）
- 警惕"政策受益"变"政策依赖"——当补贴/扶持成为盈利主要来源时，本身就是风险
- 两会、政治局会议、中央经济工作会议是重要的政策信号窗口期
