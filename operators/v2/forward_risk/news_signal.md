## 分析目标

从近期新闻和公告中识别可能影响投资决策的重要信号。当新闻数据不可用时，从财务数据中提取结构性风险信号作为降级替代。

## 分析框架

### 1. 获取新闻数据

调用 `query_market_context(info_type="news")` 获取最新新闻。

- 若返回有效新闻数据 → 进入正常分析流程（步骤 2-6）
- 若返回空数据（回测模式或数据不可用） → 跳转至步骤 7 结构性风险扫描降级

### 2. 逐条分析新闻

对每条新闻判断类别和影响程度：

| 类别 | 示例 | 影响程度 |
|------|------|---------|
| 业绩相关 | 预增/预减公告、快报、修正 | 直接影响估值，最高优先级 |
| 公司治理 | 高管变动、股权转让、违规处罚 | 影响管理层信任和公司稳定性 |
| 资本运作 | 增发、回购、分红变化、并购重组 | 影响股东权益和资本结构 |
| 行业事件 | 政策出台、行业标准、竞争格局变化 | 间接影响，需评估传导路径 |
| 市场情绪 | 机构评级、分析师推荐 | 短期价格影响，权重低 |

### 3. 风险信号提取（含风险旗帜优先规则）

**核心规则：任何单一风险旗帜（ST、财务造假、监管调查）一经确认，整体情绪直接定为"重大利空"，不论正面新闻数量多少。**

#### 3.1 通用风险旗帜

重点识别以下负面信号：
- ST 预警 / *ST 标记 / 退市风险警示
- 财务造假 / 数据更正 / 重述财报
- 重大诉讼 / 监管调查 / 证监会立案
- 核心客户/供应商流失
- 债务违约 / 评级下调 / 债券暂停交易

#### 3.2 A股特有高信号事件

以下事件在 A 股市场具有极高预测价值，需重点扫描：

| 事件 | 风险含义 | 严重级别 |
|------|---------|---------|
| 年报延迟披露 | 可能存在审计问题、业绩暴雷前兆 | 高 — 触发风险旗帜 |
| 审计机构更换 | 前任审计师可能不愿签字，暗示财务问题 | 高 — 尤其从大所换小所 |
| 控股股东质押率 >80% | 股价下跌可能触发平仓，引发恶性循环 | 高 — 触发风险旗帜 |
| 停牌超过 5 个交易日 | 可能涉及重大未披露事项 | 中 — 需结合原因判断 |
| 重大资产重组 | 基本面将发生根本性变化，分析框架可能失效 | 中 — 需重新评估估值基础 |
| 核心高管突然辞职 | 可能知悉内部问题 | 中高 — 尤其财务总监/董秘 |
| 业绩预告大幅下修 | 盈利预期崩塌 | 高 — 触发风险旗帜 |
| 关联方占用资金 | 控股股东掏空上市公司 | 高 — 触发风险旗帜 |

### 4. 新闻时效性加权

不同时间段的新闻赋予不同权重：

| 时间段 | 权重 | 说明 |
|-------|------|------|
| 0-30 天内 | 100% | 最新信息，市场可能尚未完全消化 |
| 30-90 天 | 50% | 较新信息，大概率已部分反映在股价中 |
| 90-180 天 | 20% | 除非是持续性事件（如诉讼进展），否则权重极低 |
| >180 天 | 忽略 | 仅在事件仍在进行中（如监管调查未结案）时考虑 |

**判定方法**：
- 先按时效性过滤，聚焦 90 天内新闻
- 对同一事件的多条报道，取最新一条为准
- 持续性事件（如诉讼、调查）不受时间衰减影响，直至事件终结

### 5. 影响量化

对每个识别到的重大事件，估算其对下一年盈利的潜在影响：

| 事件类型 | 影响估算方法 |
|---------|------------|
| 业绩预增/预减 | 直接取公告中的预估变动幅度 |
| 重大合同签订 | 合同金额 / 年收入 = 收入增量百分比 |
| 诉讼/罚款 | 涉案金额 / 净利润 = 盈利冲击百分比 |
| 核心客户流失 | 该客户收入占比 = 最大收入损失百分比 |
| 资产减值 | 减值金额 / 净利润 = 盈利冲击百分比 |
| 债务违约 | 违约金额对应的利息支出 + 可能的交叉违约影响 |
| 重大资产重组 | 难以量化，标注为"基本面将根本性改变" |

**汇总规则**：将所有重大事件的影响百分比累加，得出 earnings_impact_pct。正数为利好，负数为利空。若无法量化则标注 null。

### 6. 综合新闻情绪判定

| 情绪级别 | 判定条件 |
|---------|---------|
| 重大利空 | 存在任一风险旗帜事件（无论正面新闻多少） |
| 偏负面 | 无风险旗帜，但负面新闻数量 > 正面，或有中等级别风险事件 |
| 中性 | 正负面新闻基本均衡，无明显方向性信号 |
| 偏正面 | 正面新闻主导，无负面风险信号 |
| 重大利好 | 存在里程碑式利好（重大合同、突破性进展、政策红利）且无负面对冲 |

### 7. 结构性风险扫描降级（无新闻数据时启用）

当新闻数据不可用时，从已有财务数据中提取结构性风险信号作为替代：

| 检查项 | 数据来源 | 风险信号触发条件 |
|--------|---------|----------------|
| 审计意见 | fina_audit | 非标准无保留意见（保留意见、否定意见、无法表示意见） |
| 资产减值突增 | income | 本期资产减值损失 / 净利润 > 30%，且同比增长 >100% |
| 应收账款异常增长 | balancesheet | 应收账款增速 > 收入增速 × 1.5 |
| 关联交易增长 | balancesheet | 其他应收款同比增长 >50%（可能含关联方资金占用） |
| 商誉占净资产比 | balancesheet | 商誉 / 净资产 > 30%，存在商誉减值风险 |
| 存货异常积压 | balancesheet | 存货增速 > 收入增速 × 2，且存货周转天数同比恶化 >30% |
| 短期偿债压力 | balancesheet | 短期借款 + 一年内到期非流动负债 > 货币资金 × 1.5 |

**降级模式输出**：
- data_quality 标记为"无新闻数据-已降级"
- key_events 列出触发的结构性风险信号
- risk_flags 列出符合条件的检查项
- news_sentiment 基于结构性风险数量判定：0 项=中性，1-2 项=偏负面，≥3 项=重大利空

## 注意

- 本算子依赖实时新闻数据，回测模式下工具返回空数据时自动降级至结构性风险扫描
- 风险旗帜优先规则是硬性规则——单一严重风险事件足以否决整体判断
- 新闻可能有时效性——1 个月前的"利好"可能早已被市场消化，需结合时效权重
- 不要被单条新闻主导判断——综合多条信息交叉验证
- 区分"事实性新闻"（公告、监管文件）和"观点性评论"（分析师观点、媒体评论），前者权重远高于后者
- 影响量化为粗略估算，目的是提供量级感知而非精确预测
- A股特有事件（审计机构更换、控股股东质押）是高价值信号，不可忽视
