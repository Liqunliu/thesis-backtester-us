## 周期位置判断与拐点分析

本算子用于判断目标公司所处行业的周期位置，评估是否处于周期底部/顶部，以及在周期底部是否可以容忍较高负债水平。适用于周期性行业（乳业、化工、航运、钢铁、有色、煤炭、养殖、半导体等）。

### 一、周期位置三维判定框架

#### 1. 价格维度

- 获取该行业核心产品/商品的价格数据（如奶价、煤价、铜价、猪价、运价指数等）
- 计算当前价格在近10年中的百分位数

| 价格百分位 | 周期位置判断 | 含义 |
|-----------|-------------|------|
| 0%-20% | 深度底部 | 行业严重亏损，产能出清中 |
| 20%-40% | 底部区域 | 盈利压缩，弱势企业退出 |
| 40%-60% | 中部区域 | 正常盈利水平 |
| 60%-80% | 景气区域 | 盈利扩张，产能扩张启动 |
| 80%-100% | 顶部区域 | 超额利润，需警惕回调 |

#### 2. 产能维度

- 行业产能利用率：< 70% 为底部区域，> 85% 为景气区域
- 行业资本开支趋势：连续2年下降 -> 产能收缩中（底部信号）
- 行业内企业数量变化：龙头市占率提升 -> 尾部出清中

#### 3. 库存维度

- 渠道库存水平：高库存 + 主动去库 -> 仍在筑底
- 企业原材料库存：企业开始补库 -> 需求恢复信号
- 产成品库存：连续下降 -> 去库存接近尾声

### 二、综合周期位置判定

```
三维度汇总：
  价格处于10年 [__]% 分位
  产能利用率约 [__]%
  库存处于 [去库/补库/被动累库/主动累库] 阶段

判定逻辑：
  三维度均指向底部 -> 确认周期底部
  两个维度指向底部 -> 大概率底部，需进一步验证
  一个维度指向底部 -> 周期位置模糊，保守处理
  无维度指向底部 -> 非周期底部
```

### 三、周期底部容忍高负债的条件

在确认周期底部后，若公司负债偏高，需满足以下条件中的至少4项方可接受：

| 序号 | 判定条件 | 具体标准 | 通过? |
|------|----------|----------|-------|
| 1 | 经营现金流为正且改善 | 同比增长 > +30% | [ ] |
| 2 | 利息覆盖 | 经营现金流 > 利息支出的2倍 | [ ] |
| 3 | 主动去杠杆 | 有息负债绝对额同比下降 | [ ] |
| 4 | 资产处置回收现金 | 出售非核心资产，金额可观 | [ ] |
| 5 | 管理层降本动作 | 费用率下降或明确的削减计划 | [ ] |
| 6 | 股东背景兜底 | 央国企/大型集团背景，融资渠道畅通 | [ ] |

通过4项及以上 -> 可接受适度高负债（但需设置负债改善的跟踪时间表）
通过2-3项 -> 谨慎，需密切跟踪
通过0-1项 -> 不接受高负债，回避

### 四、周期拐点信号检测

#### 正向拐点信号（底部反转）

```
[ ] 产品价格企稳并出现首次环比上涨
[ ] 行业内出现大规模减产/停产公告
[ ] 龙头企业资本开支计划开始回升
[ ] 下游需求出现边际改善（订单/开工率）
[ ] 企业原材料采购量增加（主动补库信号）
[ ] 经营现金流由负转正或大幅改善
```

#### 负向拐点信号（顶部回落）

```
[ ] 产品价格创新高但涨幅放缓
[ ] 行业大量新增产能即将投产
[ ] 下游需求增速放缓或出现去库存
[ ] 龙头企业开始谨慎表态
[ ] 政策面出现抑制过热的信号
```

### 五、非现金项目还原

周期底部公司利润表往往受大额非现金项目扭曲，需还原真实盈利能力：

| 项目 | 性质 | 调整方式 | 说明 |
|------|------|----------|------|
| 资产减值损失 | 非现金 | 加回利润 | 周期底部常见大额计提，不影响现金流 |
| 固定资产折旧 | 非现金 | 加回利润 | 计算EBITDA时加回 |
| 无形资产/商誉摊销 | 非现金 | 加回利润 | 同上 |
| 汇兑损益 | 非经营 | 单独列示 | 不反映主业经营能力 |
| 公允价值变动 | 非现金 | 单独列示 | 投资性房地产/金融资产重估 |
| 存货跌价准备 | 非现金 | 加回利润 | 周期回升时可能转回 |

**还原公式**：
调整后利润 = 报表净利润 + 资产减值 + 折旧摊销 +/- 汇兑损益 +/- 公允价值变动

### 六、分析输出模板

1. **行业周期位置**：[底部/中部/顶部]，基于价格[__]%分位 + 产能利用率[__]% + 库存[阶段]
2. **拐点信号**：[有/无]明确拐点信号，具体表现为[...]
3. **负债容忍度评估**（仅周期底部适用）：满足[N]/6项条件，[可接受/谨慎/回避]
4. **非现金项目还原**：报表利润[X]亿，还原后利润[Y]亿，差异原因[...]
5. **跟踪建议**：下一步需关注的周期指标和时间节点
