## 周期位置判断与拐点分析

本算子用于判断目标公司所处行业的周期位置，评估是否处于周期底部/顶部，以及在周期底部是否可以容忍较高负债水平。适用于周期性行业（乳业、化工、航运、钢铁、有色、煤炭、养殖、半导体等）。

### 数据回退方案

本算子理想情况下需要行业商品价格、产能利用率、渠道库存等外部数据，但这些数据在当前数据层中不可用。以下为基于财务报表的替代方案：

| 理想数据 | 替代指标（财报可得） | 数据来源 | 说明 |
|----------|---------------------|----------|------|
| 商品价格百分位 | **毛利率百分位**（取可获取的最长历史） | income | 毛利率处于历史低位 ≈ 产品价格处于低位 |
| 产能利用率 | **资本开支/折旧比**（capex÷折旧摊销） | cashflow, balancesheet | 比值<1表示净缩减产能（底部信号），>2表示大幅扩产（景气/顶部信号） |
| 渠道库存水平 | **存货周转率变化**（同比） | balancesheet, income | 周转率下降=库存积压，周转率回升=去库接近尾声 |
| 10年价格历史 | 若不足10年数据，用可获取的最长历史（通常2015年至今） | — | 分位数仍有效，但需注明样本期长度 |

**重要**：当外部行业数据不可用时，所有三个维度均改用财报替代指标，并在输出中注明"基于财报替代指标判断"。

### 一、周期位置三维判定框架

#### 1. 价格维度（财报替代：毛利率分位）

- 计算公司毛利率（销售毛利率 = (营收-营业成本)/营收）的历史序列
- 计算当前毛利率在可获取最长历史中的百分位数（若不足10年数据，用可获取的最长历史，并注明样本期）
- 同时参考 fina_indicator 中的销售毛利率（gross_profit_margin）交叉验证

| 毛利率百分位 | 周期位置判断 | 含义 |
|-------------|-------------|------|
| 0%-20% | 深度底部 | 行业严重亏损，产能出清中 |
| 20%-40% | 底部区域 | 盈利压缩，弱势企业退出 |
| 40%-60% | 中部区域 | 正常盈利水平 |
| 60%-80% | 景气区域 | 盈利扩张，产能扩张启动 |
| 80%-100% | 顶部区域 | 超额利润，需警惕回调 |

#### 2. 产能维度（财报替代：资本开支/折旧比）

- 从现金流量表取"购建固定资产、无形资产和其他长期资产支付的现金"作为资本开支
- 从利润表/资产负债表取折旧摊销额
- 计算 capex/折旧比（资本开支÷折旧摊销）

| capex/折旧比 | 产能方向判断 |
|-------------|-------------|
| < 0.8 | 产能净收缩（底部信号） |
| 0.8 - 1.2 | 维持性资本开支（中性） |
| 1.2 - 2.0 | 温和扩产 |
| > 2.0 | 大幅扩产（景气区域或过热信号） |

- 辅助验证：连续2年 capex/折旧比 < 1 -> 产能收缩中（底部信号）
- 辅助验证：固定资产同比增速放缓或下降 -> 行业扩张放缓

#### 3. 库存维度（财报替代：存货变动+营收变动组合判断）

利用资产负债表存货数据和利润表营收数据判断库存周期阶段：

| 产成品存货同比 | 营收同比 | 库存周期阶段 | 含义 |
|---------------|---------|-------------|------|
| ↑ 上升 | ↓ 下降 | **被动累库** | 需求萎缩，库存被动堆积（衰退期） |
| ↓ 下降 | ↓ 下降 | **主动去库** | 企业主动减产去库存（底部区域） |
| ↓ 下降 | ↑ 上升 | **被动去库** | 需求恢复快于补库（复苏信号） |
| ↑ 上升 | ↑ 上升 | **主动补库** | 企业主动补库存（景气上行） |

- 辅助验证：存货周转率连续改善 -> 去库接近尾声
- 辅助验证：存货占总资产比重变化趋势

### 二、综合周期位置判定

```
三维度汇总（基于财报替代指标）：
  毛利率处于历史 [__]% 分位（样本期：[YYYY]年至今）
  capex/折旧比约 [__]（连续[N]年趋势：[上升/持平/下降]）
  库存处于 [被动累库/主动去库/被动去库/主动补库] 阶段

判定逻辑：
  三维度均指向底部 -> 确认周期底部
  两个维度指向底部 -> 大概率底部，需进一步验证
  一个维度指向底部 -> 周期位置模糊，保守处理
  无维度指向底部 -> 非周期底部
```

### 三、周期底部容忍高负债的条件

在确认周期底部后，若公司负债偏高，需满足以下条件中的至少4项方可接受：

| 序号 | 判定条件 | 具体标准 | 数据来源 | 通过? |
|------|----------|----------|----------|-------|
| 1 | 经营现金流为正且改善 | 同比增长 > +30% | cashflow | [ ] |
| 2 | 利息覆盖 | 经营现金流 > 利息支出的2倍 | cashflow, income | [ ] |
| 3 | 主动去杠杆 | 有息负债绝对额同比下降 | balancesheet | [ ] |
| 4 | 资产处置回收现金 | 处置固定资产等收回的现金净额显著 | cashflow | [ ] |
| 5 | 管理层降本动作 | 管理费用率+销售费用率同比下降 | income | [ ] |
| 6 | 股东背景兜底 | 央国企/大型集团背景，融资渠道畅通 | basic_info | [ ] |

通过4项及以上 -> 可接受适度高负债（但需设置负债改善的跟踪时间表）
通过2-3项 -> 谨慎，需密切跟踪
通过0-1项 -> 不接受高负债，回避

### 四、周期拐点信号检测（财报可识别信号）

#### 正向拐点信号（底部反转）

```
[ ] 毛利率环比回升（连续2个季度）
[ ] 存货同比下降但营收企稳或回升（被动去库 → 复苏信号）
[ ] capex/折旧比触底回升
[ ] 经营现金流由负转正或大幅改善
[ ] 应收账款周转改善（下游付款意愿增强）
[ ] 资产减值损失大幅缩小（最坏阶段已过）
```

#### 负向拐点信号（顶部回落）

```
[ ] 毛利率环比走平或下降（连续2个季度）
[ ] 存货同比快速增长但营收增速放缓（被动累库）
[ ] capex/折旧比 > 2 且持续上升（产能过度扩张）
[ ] 应收账款增速显著超过营收增速（下游资金紧张）
[ ] 行业内同行报表均显示类似产能扩张
```

### 五、非现金项目快速估算

> 详细利润还原请参考 `performance_restoration` 算子。此处仅提供周期语境下的快速估算方法：
> 调整后利润 ≈ 报表净利润 + 资产减值 + 折旧摊销
> 注意：此为粗略估算，用于判断周期位置，不用于估值。

### 六、分析输出模板

1. **行业周期位置**：[底部/中部/顶部]，基于毛利率历史[__]%分位（样本期[N]年） + capex/折旧比[__] + 库存阶段[被动累库/主动去库/被动去库/主动补库]
2. **拐点信号**：[有/无]明确拐点信号，具体表现为[...]
3. **负债容忍度评估**（仅周期底部适用）：满足[N]/6项条件，[可接受/谨慎/回避]
4. **非现金项目还原**：报表利润[X]亿，还原后利润[Y]亿，差异原因[...]（快速估算，详见 performance_restoration 算子）
5. **数据局限说明**：本次分析基于[财报替代指标/外部行业数据]，样本期[N]年，[是否存在]数据不足的维度
6. **跟踪建议**：下一步需关注的周期指标和时间节点
