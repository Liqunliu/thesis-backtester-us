## 业绩还原与投资视角分离框架

将报表分析拆分为两个独立维度：客观的业绩还原（真实经营利润是多少）和主观的投资决策（基于真实利润该如何行动）。两者分离，避免因报表失真导致错误决策。

### 一、核心原则

```
报表净利润 ≠ 真实经营利润
真金白银现金流 > 报表利润数字

分析必须回答两个问题：
1. 业绩还原（客观）：这家公司实际赚了多少钱？
2. 投资视角（主观）：基于真实盈利，当前价格值不值得买？
```

### 二、维度一：业绩还原（报表利润→真实经营利润）

#### 2.1 还原公式

```
报表净利润：[X]亿元（可能受非现金项目严重扭曲）

加回非现金减值项：
  + 商誉减值：[A]亿元（收购产生的一次性/多次计提）
  + 资产减值：[B]亿元（固定资产、长期股权投资减值）
  + 存货减值：[C]亿元（库存跌价准备）

加回非现金费用：
  + 折旧摊销：[D]亿元（非现金支出，尤其重资产行业）
  + 股权激励费用：[E]亿元（非现金薪酬）

扣除非经常性收益：
  - 投资收益（一次性）：[F]亿元（处置资产/股权）
  - 政府补贴（不可持续部分）：[G]亿元

扣除汇兑损益：
  - 汇兑损益：[H]亿元（非经营性波动）

═══════════════════════════════════
= 实际经营利润：[还原值]亿元
  同比增长：[计算同比]（基于还原后口径对比）
```

#### 2.2 需要还原的典型行业

| 行业 | 常见失真项 | 还原重点 |
|------|-----------|---------|
| 乳业/快消 | 大额折旧 + 商誉减值（海外收购） | 加回非现金减值，看经营现金流 |
| 制造业 | 重资产折旧大、存货减值波动 | 区分维持性CAPEX与扩张性CAPEX |
| 科技 | 股权激励费用、研发资本化 | 加回股权激励，扣除资本化部分 |
| 地产 | 存货减值、合营利润波动 | 穿透合营项目，看结算节奏 |
| 金融 | 拨备计提波动 | 看拨备前利润和拨备覆盖率 |

#### 2.3 还原质量验证

还原后应通过以下交叉验证：

```
验证1：现金流一致性
  还原后经营利润 vs 经营现金流
  ├── 比值在0.8-1.2之间 → 还原合理
  ├── 比值<0.8 → 还原可能不足，仍有隐藏失真
  └── 比值>1.2 → 还原可能过度，检查是否加回了真实损失

验证2：趋势一致性
  还原后利润趋势 vs 收入趋势
  ├── 同向变动 → 合理
  └── 背离 → 需检查利润率变化原因

验证3：同行对比
  还原后利润率 vs 同行业公司
  ├── 差异<5个百分点 → 合理
  └── 差异过大 → 需重新审视还原逻辑
```

### 三、维度二：投资视角（真实利润→投资决策）

#### 3.1 现金流评估

```
年进账现金（经营现金流）：[X]亿元 → 真实造血能力
- 维持性CAPEX：[Y]亿元
= 自由现金流（FCF）：[X-Y]亿元 → 可分配给股东的真金白银
FCF/市值：[Z]% → 真实投资回报率

判定标准：
├── FCF/市值 > 20% → 极具吸引力
├── FCF/市值 10-20% → 有吸引力
├── FCF/市值 5-10% → 一般
└── FCF/市值 < 5% → 缺乏安全边际
```

#### 3.2 安全边际评估

```
基于还原后真实利润的估值：
├── 还原后PE = 市值 / 实际经营利润
├── 剔除净现金FCF倍数 = (市值-净现金) / FCF
└── 真实股息支付能力 = FCF / 当前股息支出

价格位置判断：
├── 距52周低点 [X]%（>30%则需谨慎）
├── 距历史估值中位数 [X]%
└── 当前安全边际：[充足/一般/不足]

行动决策：
├── 安全边际充足 + 价格位置好 → 建仓
├── 安全边际充足 + 价格偏高 → 等待回调（设定触发价）
├── 安全边际不足 → 放弃或仅观察
└── 关键纪律：不因业绩好就买入，要看价格是否匹配
```

### 四、双维度分析流程

```
开始分析标的
    |
    v
步骤1：业绩还原（客观真相）
    ├── 收集报表数据（净利润、现金流、资产负债表）
    ├── 识别非现金项目（减值、折旧、激励费用）
    ├── 识别非经常性损益
    ├── 还原实际经营利润
    ├── 交叉验证还原结果
    └── 输出：实际经营利润 [X]亿元，同比 [Y]%
    |
    v
步骤2：投资视角（主观决策）
    ├── 计算真金白银现金流（FCF）
    ├── 计算还原后估值指标（还原PE、FCF倍数）
    ├── 评估当前价格安全边际
    ├── 对比其他标的性价比
    └── 决策：买入 / 等待回调至[触发价] / 放弃
    |
    v
步骤3：形成投资结论
    ├── 一句话逻辑：[还原后利润情况] + [价格匹配度] + [行动方案]
    ├── 买入触发价：[基于还原利润的合理估值倒算]
    └── 仓位上限：[根据确定性决定]
```

【投资视角】
