## 估值修复框架与目标价设定

### 一、核心理念

估值修复(valuation re-rating)是价值投资的重要收益来源之一。本框架帮助分析师区分"修复型"与"成长型"投资逻辑，设定合理目标价，并通过反"永远持有"原则强制执行退出纪律。

**三类收益来源**：
1. 股息收益（持有期现金回报）
2. 估值修复（市场重新定价，如EV/FCF从3倍修复到8倍）
3. 盈利增长（基本面改善推动内在价值提升）

### 二、投资逻辑分流

不同标的的主要收益来源不同，需明确分流后设定不同的目标价逻辑：

| 投资逻辑类型 | 特征 | 主要收益来源 | 目标价锚定 |
|------------|------|------------|-----------|
| 纯修复型 | 被错杀/被忽视，基本面稳定 | 估值修复（主）+ 股息 | 合理估值倍数还原 |
| 修复+成长型 | 低估且有增长动力 | 估值修复 + 盈利增长 | 合理倍数 x 增长后盈利 |
| 纯成长型 | 估值合理但增长强劲 | 盈利增长（主） | 当前倍数 x 增长后盈利 |
| 高股息收息型 | 稳定分红，估值长期低迷 | 股息收益（主） | 股息率回归历史中位 |

### 三、估值修复目标价设定方法

#### 3.1 基于EV/FCF的修复目标

```
修复目标价计算：

当前状态：
- FCF = [ ]亿元
- 当前EV/FCF = [ ]倍
- 当前市值 = [ ]亿元
- 净现金 = [ ]亿元

修复目标（不同情景）：

保守修复（合理估值下限）：
- 目标EV/FCF = 8倍
- 目标EV = FCF x 8 = [ ]亿元
- 目标市值 = 目标EV + 净现金 = [ ]亿元
- 目标价 = 目标市值 / 总股本 = [ ]元
- 上行空间 = (目标价 - 现价) / 现价 = [ ]%

中性修复（行业合理中枢）：
- 目标EV/FCF = 10倍
- （同上计算流程）

乐观修复（牛市环境）：
- 目标EV/FCF = 12倍
- （同上计算流程）
```

#### 3.2 基于PE的修复目标（适用行业）

```
目标PE选择参考：
- 行业历史PE中位数
- 可比公司当前PE中位数
- 公司自身历史PE中位数（排除极端值）

修复目标市值 = 归母净利润 x 目标PE
```

#### 3.3 基于股息率的修复目标

```
目标股息率 = 历史股息率中位数（或无风险利率x1.5）
修复目标价 = 每股股息 / 目标股息率
```

### 四、修复型 vs 成长型：目标价差异

| 项目 | 修复型目标价 | 成长型目标价 |
|-----|-----------|-----------|
| 盈利假设 | 使用当前FCF/盈利（不假设增长） | 使用未来2-3年预期盈利 |
| 倍数假设 | 从当前低估倍数修复到合理区间 | 维持当前倍数或小幅提升 |
| 时间框架 | 1-2年（市场重新认知） | 2-3年（盈利兑现） |
| 风险点 | 修复催化剂不出现 | 增长不及预期 |
| 退出信号 | 达到合理估值即减仓 | 增长逻辑被证伪 |

**修复型标的的典型催化剂**：提高派息比例、回购计划、纳入指数、分析师覆盖增加、行业政策利好、同行估值带动。

### 五、反"永远持有"原则

**核心观点**：没有任何股票值得"永远持有"。每个买入都必须有明确的退出条件和目标价。

#### 5.1 强制退出规则

```
退出决策树：

检查1：是否达到修复目标价？
    +-- 是 -> 至少减仓50%
    +-- 否 -> 继续持有，检查下一项

检查2：持仓盈利是否超过50%？
    +-- 是 -> 强制减仓50%（锁定利润）
    +-- 否 -> 继续持有

检查3：EV/FCF是否超过合理区间上限？
    +-- 牛市中>8倍 -> 减仓
    +-- 任何时期>12倍 -> 清仓
    +-- 否 -> 继续持有

检查4：基本面是否恶化？
    +-- FCF连续2季下滑>20% -> 减仓
    +-- 竞争格局恶化 -> 减仓
    +-- 管理层变动 -> 重新评估
    +-- 否 -> 继续持有

检查5：是否持有超过3年未达修复目标？
    +-- 是 -> 重新评估投资逻辑
    +-- 修复逻辑仍成立 -> 可继续，但需更新催化剂
    +-- 修复逻辑失效 -> 止损退出
```

#### 5.2 分批退出方案

| 触发条件 | 退出比例 | 剩余处理 |
|---------|---------|---------|
| 达到保守目标 | 卖出30% | 止损线提至成本价 |
| 达到中性目标 | 再卖30% | 止损线提至保守目标 |
| 达到乐观目标 | 再卖30% | 剩余10%观察 |
| 超过乐观目标 | 清仓 | -- |

### 六、目标价合理性检验

```
[ ] 隐含年化回报率>15%？（低于此不值得承担股票风险）
[ ] 隐含估值倍数在行业合理区间内？
[ ] 目标估值低于可比公司当前估值？（修复不应假设超越同行）
[ ] 盈利下降20%时，目标价对应EV/FCF仍可接受？
[ ] 催化剂明确且概率可评估？无催化剂时有退出时限？
```

### 七、完整评估模板

```
估值修复分析：
1. 投资逻辑分类：[纯修复/修复+成长/纯成长/高股息收息]
2. 当前估值：EV/FCF = [ ]倍，PE = [ ]倍，股息率 = [ ]%
3. 修复目标价（保守）：[ ]元（EV/FCF [ ]倍），上行空间[ ]%
4. 修复目标价（中性）：[ ]元（EV/FCF [ ]倍），上行空间[ ]%
5. 修复目标价（乐观）：[ ]元（EV/FCF [ ]倍），上行空间[ ]%
6. 预期催化剂：[ ]
7. 催化剂出现概率：[高/中/低]
8. 退出规则：达到[ ]目标减仓[ ]%
9. 止损条件：[ ]
10. 最长持有期限：[ ]年
11. 隐含年化回报率：[ ]%
12. 目标价合理性检验：[通过/未通过]
```
