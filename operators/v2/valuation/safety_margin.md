## 动态安全边际（市场周期联动）

### 一、核心原则

安全边际不是固定值，应根据市场整体估值水平（牛熊周期）动态调整。牛市放宽入场标准但收紧退出纪律，熊市收紧入场标准但获得更高补偿。

### 二、市场周期判定标准

| 市场状态 | 判定条件 |
|---------|---------|
| 牛市 | 较52周低点上涨>30%，且成交量持续放大 |
| 震荡市 | 较52周低点上涨10-30%，或窄幅波动 |
| 熊市 | 较52周高点下跌>20%，或破净个股比例>30% |

**辅助判定指标**：
- AH溢价指数 > 130 表示港股相对便宜
- 全市场PE中位数处于历史低位区间

**市场周期判断增强**：
除52周高低点外，综合使用以下信号：
1. 指数PE历史百分位（vs 10年历史）
2. 融资余额趋势（上升=偏牛，下降=偏熊）
3. 全市场涨跌比（涨停/跌停家数比）
4. 新增开户数趋势

> 单一的52周高低点是滞后指标，容易误判。多信号综合判断更可靠。

### 三、差异化安全边际标准

| 估值指标 | 牛市标准 | 震荡市标准 | 熊市标准 |
|---------|---------|-----------|---------|
| EV/FCF倍数（核心） | < 5倍 | < 4倍 | < 3倍 |
| PE（剔除净现金） | < 10倍 | < 8倍 | < 6倍 |
| PB | < 1.0倍 | < 0.8倍 | < 0.6倍 |
| 股息率 | > 3.8% | > 4.5% | > 6% |
| FCF/市值 | > 8% | > 10% | > 12% |
| 净现金/市值 | > 30% | > 40% | > 50% |

**解读**：
- 牛市时估值中枢上移，以EV/FCF<5倍为可建仓线
- 熊市时估值中枢下移，要求EV/FCF<3倍才可建仓（极度低估区间）
- 震荡市取中间值

### 四、动态调整执行流程

```
步骤1：判定市场周期
    +-- 查看主要指数52周高低点位置
    +-- 查看破净个股比例
    +-- 查看AH溢价指数（港股适用）
    +-- 判定结果：[牛市/震荡市/熊市]

步骤2：选择对应安全边际标准
    +-- 牛市 -> 执行牛市标准
    +-- 震荡市 -> 执行中间标准
    +-- 熊市 -> 执行熊市标准

步骤3：逐项检查标的估值
    +-- EV/FCF倍数 = [ ]倍 -> [达标/未达标]
    +-- PE（剔除净现金）= [ ]倍 -> [达标/未达标]
    +-- PB = [ ]倍 -> [达标/未达标]
    +-- 股息率 = [ ]% -> [达标/未达标]
    +-- FCF/市值 = [ ]% -> [达标/未达标]
    +-- 达标项数：[ ]/5

步骤4：综合判定
    +-- 核心指标（EV/FCF）必须达标
    +-- 辅助指标至少3项达标
    +-- 全部达标 -> 高安全边际
    +-- 核心+2-3项 -> 一般安全边际
    +-- 核心未达标 -> 安全边际不足

步骤5：定期重估（月度）
    +-- 市场上涨>10% -> 收紧买入标准（要求更低估值才建仓，即向熊市标准方向调整）
    +-- 市场下跌>10% -> 可适度放宽标准（估值已系统性压缩，即向牛市标准方向调整）
    +-- 核心逻辑：市场越热越谨慎，市场越冷越大胆
```

### 五、牛市特别风险警示

牛市是最危险的时刻。以下风险信号需要强制检查：

```
牛市陷阱检查清单：

[ ] "这次不一样"思维
    -> 对策：坚持量化标准，不因市场情绪放宽

[ ] 降低标准买入
    -> 对策：牛市末期买入等于站岗，不追涨

[ ] 忽视股息率
    -> 对策：牛市中股息率跌破3%风险极高

[ ] 忘记动态退出
    -> 对策：牛市中更应严格执行退出纪律

牛市自我保护规则：
- 坚持PE硬标准（如<10倍），不因上涨放宽至15倍
- 坚持股息率保底收益，不因成长故事放弃
- 持仓上涨50%强制减仓50%
- 保留至少20%现金等待回调
- 提高退出阈值（如EV/FCF>8倍即减仓）
```

### 六、仓位与安全边际联动

| 安全边际等级 | EV/FCF位置 | 建议仓位 |
|------------|-----------|---------|
| 极高（远超标准） | 牛市<3倍 / 熊市<2倍 | 极限仓位（如15%） |
| 高（达标） | 牛市3-5倍 / 熊市2-3倍 | 重仓（10-12%） |
| 一般（刚达标） | 牛市5-8倍 / 熊市3-5倍 | 正常仓位（8%） |
| 不足（未达标） | 超过对应标准 | 不建仓或观望 |

**价格位置调节**：即使估值达标，若股价已从低点大幅上涨（如>50%），需降低一档仓位。安全边际同时考虑估值水平和价格位置。
