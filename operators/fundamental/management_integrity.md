## 管理人诚信与治理风险评估

本算子用于系统评估上市公司管理层的诚信度和公司治理质量。管理人诚信是价值投资的前提条件 -- 再便宜的公司，如果管理层不可信，投资安全边际也为零。

### 一、评估维度与检查项

#### 维度1：承诺兑现（权重30%）

```
检查清单：
  [ ] 过去3年公开增持承诺是否全部执行？
  [ ] 回购计划是否按期完成？（金额、时间）
  [ ] 业绩指引/展望的准确度？（偏差 < 15%为合格）
  [ ] 分红政策是否稳定执行？（连续性、派息率）
  [ ] 战略规划是否有持续推进？（非空头支票）

评分：
  全部兑现或偏差 < 10% -> 优
  大部分兑现，偏差 10%-30% -> 良
  部分未兑现，偏差 > 30% -> 差
  频繁食言或反复修改目标 -> 极差
```

#### 维度2：关联交易与利益输送（权重30%）

```
检查清单：
  [ ] 关联交易总额占营收比例？
      < 10%: 正常   10%-30%: 关注   > 30%: 高风险
  [ ] 关联交易定价是否公允？（与市场价对比）
  [ ] 应收关联方款项金额及账龄？
      长期大额应收关联方 -> 资金可能被占用
  [ ] 是否存在与管理层个人关联的交易？
  [ ] 担保结构：是否为关联方提供大额担保？
  [ ] 资金往来：是否存在频繁的资金拆借？
```

**重点关注的资金操纵手法**：
| 手法 | 识别特征 | 风险等级 |
|------|----------|----------|
| 大存大贷 | 高额存款+高额借款并存，存款可能受限 | 高 |
| 体外循环 | 通过关联方转移资金再回流制造收入 | 极高 |
| 份额支付 | 用股份代替现金支付管理费/对价 | 高 |
| 隐性担保 | 表外为关联方提供担保未披露 | 高 |

#### 维度3：信息披露质量（权重20%）

```
检查清单：
  [ ] 年报篇幅与详细程度？（与同行对比）
  [ ] 风险因素披露是否具体？（泛泛而谈 vs 针对性说明）
  [ ] 会计政策是否频繁变更？（变更理由是否充分）
  [ ] 审计意见类型？
      标准无保留 -> 正常
      带强调事项段 -> 需关注
      保留意见/否定意见 -> 高风险
  [ ] 是否更换过审计师？（频繁更换 -> 警惕）
  [ ] 内控报告是否存在重大缺陷？
```

#### 维度4：减持与股权行为（权重20%）

```
检查清单：
  [ ] 大股东近3年是否有大额减持？
      股价低位减持 -> 极度负面信号
  [ ] 管理层持股变动？（期权行权后立即抛售？）
  [ ] 是否有供股/配股历史？
      过去5年供股2次以上 -> 稀释股东权益，高风险
  [ ] 股权质押比例？
      > 50% -> 高风险（平仓压力）
      > 80% -> 极高风险
  [ ] 是否存在一致行动人代持安排？
```

### 二、特殊红旗信号（一票否决项）

以下任一项出现即应高度警惕或直接回避：

```
[!] 管理层因财务造假被监管处罚
[!] 审计师出具否定意见或拒绝发表意见
[!] 大股东占用上市公司资金（经监管认定）
[!] 频繁更换CFO或审计机构（3年内2次以上）
[!] 连续供股/配股稀释小股东（5年内3次以上）
[!] 实际控制人涉及刑事案件
```

### 三、三级评级体系

#### 可信（投资前提满足）

- 承诺兑现率 > 80%
- 无重大关联交易争议
- 信息披露充分透明，标准审计意见
- 无不当减持或供股记录
- 无监管处罚记录
- 分红政策稳定且可预期

#### 存疑（需额外安全边际补偿）

- 承诺部分兑现（50%-80%）
- 存在关联交易但有合理解释
- 信息披露质量一般
- 有少量减持但时机和规模尚可
- 存在大存大贷但金额不极端
- **处置方式**：若投资，需在估值上要求额外20%-30%的安全边际

#### 不可信（建议回避）

- 承诺频繁食言（兑现率 < 50%）
- 关联交易不透明，损害中小股东利益
- 信息披露存在误导，审计意见非标
- 大股东在股价低位大量减持/高比例质押
- 有监管处罚记录或频繁供股历史
- **处置方式**：无论估值多便宜，均不投资

### 四、分析输出模板

1. **承诺兑现**：[优/良/差/极差]，具体依据[...]
2. **关联交易**：关联交易占营收[__]%，[正常/关注/高风险]，是否发现资金操纵迹象
3. **信息披露**：审计意见[类型]，披露质量[好/一般/差]
4. **股权行为**：大股东减持[有/无]，供股历史[有/无]，质押比例[__]%
5. **红旗信号**：[有/无]一票否决项
6. **综合评级**：[可信/存疑/不可信]
7. **投资建议**：基于诚信评估的处置建议
